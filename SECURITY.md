# Security Policy

## Supported versions

The public `main` branch is the supported version for this demo/reference repo.

## Reporting a vulnerability

Please email `mira.solutions06@gmail.com` with:

- The affected repo and file or endpoint.
- Steps to reproduce.
- Impact and any suggested fix.

Please do not open a public issue for a vulnerability that exposes credentials,
private media, account IDs, auth bypasses, unsafe publish behavior, or deployment
details.

## Data handling expectations

- Do not commit `.env`, local queues, generated media, access tokens, chat IDs,
  page IDs, hostnames, or account credentials.
- Keep examples and screenshots sanitized.
- Keep delivery and publishing gated by human approval.
